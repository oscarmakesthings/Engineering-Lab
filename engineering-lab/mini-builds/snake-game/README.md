# Terminal Snake Game (Go)

A fully functional Snake game written in Go using ASCII graphics.

## Features
- Real-time movement
- Score tracking
- Game timer
- Restart capability
- Cross-platform terminal support via `tcell`

## Prerequisites
- [Go](https://golang.org/dl/) (1.22 or later)

## How to Run
1. Navigate to the `snake-game` directory:
   ```bash
   cd snake-game
   ```
2. Run the game:
   ```bash
   go run main.go
   ```

## Controls
- **Movement:** Arrow Keys or WASD
- **Restart:** Enter or 'R' (on Game Over)
- **Quit:** Escape or Ctrl+C
