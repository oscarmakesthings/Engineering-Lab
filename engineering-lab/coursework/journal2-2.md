What is static testing:

Static testing is a type of software testing that is done without executing the code. It is also known as static analysis. Static testing is done at the source code level. Static testing is used to find errors in the code before it is executed.
Static testing is done by the developer or the QA team. It is done by the developer to find errors in the code before it is executed. Static testing is done by the QA team to find errors in the code after it is executed.

What is dynamic testing:

Dynamic testing is a type of software testing that is done by executing the code. It is also known as dynamic analysis. Dynamic testing is done at the code level. Dynamic testing is used to find errors in the code after it is executed.
Dynamic testing is done by the developer or the QA team. It is done by the developer to find errors in the code after it is executed. Dynamic testing is done by the QA team to find errors in the code after it is executed.
